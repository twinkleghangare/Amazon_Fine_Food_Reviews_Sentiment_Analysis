# Amazon Fine Food Reviews Sentiment Analysis

## Overview

This project analyzes customer sentiment from the Amazon Fine Food Reviews dataset using Natural Language Processing (NLP). A subset of 5,000 reviews was processed to understand customer satisfaction, identify common concerns, and extract meaningful insights from customer feedback.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* TextBlob
* NLP Techniques

## Methodology

* Cleaned the dataset by removing duplicates and missing values.
* Preprocessed review text for sentiment analysis.
* Calculated sentiment polarity scores using TextBlob.
* Classified reviews as Positive, Negative, or Neutral.
* Visualized sentiment distribution and key trends.

## Key Findings

* Approximately **78% of reviews were positive**, indicating strong customer satisfaction.
* Negative reviews were primarily associated with **delivery, packaging, and damaged products** rather than product quality.
* Common negative keywords included *broken*, *expired*, *delivery*, and *packaging*.
* Some sentiment classification errors occurred due to sarcasm and mixed opinions, highlighting limitations of rule-based sentiment analysis.

## Conclusion

The analysis shows that customers are generally satisfied with Amazon food products. Improving packaging quality and delivery processes could significantly reduce negative feedback. Future enhancements may include using advanced NLP models such as BERT for more accurate sentiment classification.
